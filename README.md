# Life Budget Planner — Legal Pages

Live site: https://reaperakrabawi.github.io/life-budget-planner/

## Enable GitHub Pages (one-time)

1. Open **Settings → Pages**: https://github.com/ReaperAkrabawi/life-budget-planner/settings/pages
2. **Build and deployment → Source**: choose **Deploy from a branch**
3. **Branch**: `main` · **Folder**: `/ (root)` · **Save**
4. Wait 2–5 minutes, then open:
   - https://reaperakrabawi.github.io/life-budget-planner/privacy.html
   - https://reaperakrabawi.github.io/life-budget-planner/terms.html

Do **not** use GitHub Actions for this repo — static HTML at the root is enough.
